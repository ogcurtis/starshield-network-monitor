# AWS Credentials Template for Starshield Network Monitor
# Copy these environment variables to your system or create a .env file

AWS_ACCESS_KEY_ID=your_access_key_here
AWS_SECRET_ACCESS_KEY=your_secret_key_here
AWS_DEFAULT_REGION=eu-central-1

# Instructions:
# 1. Get your credentials from: https://console.aws.amazon.com/iam/
# 2. Go to Users → Your User → Security credentials → Create access key
# 3. Replace the placeholder values above with your actual credentials
# 4. Set these as environment variables in your system
# 5. Or run: python scripts/setup_aws_credentials.py
